Tools Used
FastAPI
Used FastAPI to build the backend of my gender detection web application. It provides RESTful API endpoints for receiving image uploads, running the ML model inference, and returning the predicted gender to the user interface quickly and securely.
Python
Utilized Python as the primary programming language throughout the project—for data preprocessing, model development, integrating libraries, and orchestrating the workflow between the web interface, backend API, and machine learning components.
Tensorflow
Employed TensorFlow to design, train, and fine-tune the deep learning model responsible for gender prediction. The model processes input images and outputs the based-on-data prediction, forming the core logic behind the detection system.
Kaggle
Used "Gender Detection & Classification - Face Dataset" from Kaggle for training, validating, and testing the gender detection model. Kaggle’s data collections provided the variety and volume needed to achieve accurate classification results.
HTML
Developed the structure and content of the project’s web interface using HTML, allowing users to easily interact with features like uploading images, viewing results, and reading about the project tools.
CSS
Customized the appearance and layout of the website using CSS, ensuring a visually appealing, modern, and user-friendly experience while maintaining consistency with the project’s theme.
Matplotlib
Leveraged Matplotlib to visualize model performance metrics—such as confusion matrix, accuracy, and training loss curves, helping assess and present the results effectively within the project.
